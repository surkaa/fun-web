## 🛠️ 快速启动

```bash
# 推荐使用pnpm
pnpm install && pnpm dev
```

### 构建指令

| 指令             | 功能            |
|----------------|---------------|
| `pnpm dev`     | 启动开发服务器 (热更新) |
| `pnpm build`   | 生产环境构建        |
| `pnpm preview` | 本地预览生产构建      |
